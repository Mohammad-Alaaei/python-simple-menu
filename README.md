# python-simple-menu
Self made Menu for python CLI apps.

## Installation
1- first of all, open Command Prompt (windows) or Terminal (linux).

2- change directory to where your project is by CD command.
```
cd /path/to/project
```

3- activate Virtual Environment
```
.\venv\Scripts\activate
```
note: if you dont like to use Virual Environment, continue to next step.

4- install requirements
```
pip install -r requirements.txt
```

5- ready to go! follow instructions to build your first menu (instructions not ready yet, but you can learn from example.py file)
